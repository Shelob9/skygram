# Skygram dot app

[https://skgram.app](https://skgram.app) source.

## Development

- Install bun first if needed:
  - `npm i bun -g`
- Install
  - `bun i`
  - This installs web and api
- Start
  - `bun run dev`
  - This starts web and api

- [web](./skygram-web/README.md)
